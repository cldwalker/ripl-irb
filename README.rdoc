== Description
A ripl plugin that smooths the transition from irb by stubbing out any IRB method or subclass called.

== Install
Install the gem with:

    sudo gem install ripl-irb

== Usage

From the commandline

    $ ripl -rripl/irb

Or add to your ~/.riplrc

    require 'ripl/irb'
